# ✍️ Handwritten Digit Recognition using Machine Learning

## 🎯 Project Overview
This is a complete **Artificial Intelligence & Machine Learning** mini project that recognizes handwritten digits (0-9) drawn by a user using **Template Matching** algorithm.

## 🚀 Live Demo
Open `index.html` in any modern browser.

## 🎮 How to Use
1. Draw a digit (0-9) in the canvas
2. Click **"Recognize (ML)"**
3. View the predicted digit and confidence score

## 🧠 Machine Learning Techniques
- Template Matching (Instance-based learning)
- Correlation Coefficient
- Euclidean Distance
- Feature Extraction (784 pixel values)

## 💻 Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- Canvas API
